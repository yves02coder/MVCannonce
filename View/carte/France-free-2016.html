France free by c-map

Parametres, configuration

Utilisez de préférence un éditeur simple (type bloc-note, Notepad ++, komogo...) pour modifier le fichier de configuration de la carte.

  1. Ouvrez le fichier "France-map.js" contenu dans le dossier "cmap"
  2. Pour modifier les couleurs de la carte, changez les valeurs ( en rouge dans cet exemple )

    3. var mapcolor = "#70ABF3"; // couleur de fond de carte *

    var mapcolor_hover = "#FF0000"; // couleur de survol de la souris *

    var maplines = "#FFFFFF"; // couleur des traits de séparation des régions *

    trouvez facilement vos codes couleur (hexadécimaux) sur notre blog

  3. Pour chaque région, modifiez les valeurs selon vos besoins ( en rouge dans cet exemple )

    4. R0: {

    title: "Alsace",
    url: "http://mon-site.com/ma-page-Alsace.html",
    path: "269,69, 268,71, 267,73, 263,73, 258,71 ..."

    },

  4. Enregistrez et actualisez cette page d'exemple

* Pour trouver facilement vos couleurs, utilisez notre palette

 

Installation

  1. Commencez par transférer le dossier "cmap" et son contenu à la racine du serveur de votre site via ftp.
  2. Ensuite, sur la ou les pages où vous souhaitez voir la carte, ajoutez ces deux lignes de codes :
    <script src="cmap/france-map.js"></script>
    <script>francefree();</script>
